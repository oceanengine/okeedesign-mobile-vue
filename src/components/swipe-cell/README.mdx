# Swipe Cell 滑动单元格

### 安装

:::card

```javascript
import { SwipeCell } from '@okee-uikit/m-vue';

Vue.use(SwipeCell);
```

:::

### 代码演示

:::card

```topic
基础用法
```

```desc
`SwipeCell`组件提供了`left`和`right`两个插槽，用于定义两侧滑动区域的内容
```

```html
<byted-swipe-cell>
  <template slot="left">
    <byted-button text="向左" type="primary" />
  </template>

  <byted-cell title="标题" value="内容" />

  <template slot="right">
    <byted-button text="向右" type="danger" />
  </template>
</byted-swipe-cell>
```

:::

:::card

```topic
异步关闭
```

```desc
设置async，绑定close事件，可以自定义两侧滑动内容关闭时的行为
```

```html
<byted-swipe-cell async @close="onClose">
  <byted-cell title="标题" value="内容" />

  <template #right>
    <byted-button text="向右" type="danger" />
  </template>
</byted-swipe-cell>
```

```javascript
export default {
  methods: {
    // clickPosition 表示关闭时点击的位置
    onClose(clickPosition, instance) {
      switch (clickPosition) {
        case 'left':
          break;
        case 'cell':
          break;
        case 'outside':
          break;
        case 'right':
          this.$confirm('Content', 'Title')
            .then(() => {
              instance.close();
            })
            .catch(() => {
              instance.close();
            });
          break;
      }
    },
  },
};
```

:::

### API

##### Props

| 参数        | 说明                                | 类型      | 默认值  |
| ----------- | ----------------------------------- | --------- | ------- |
| name        | 标识符，可以在 close 的参数中获取到 | `string` &#124; `number` |
| async       | 关闭时的回调函数                    | `boolean` | `false` |
| disabled    | 是否禁用滑动                        | `boolean` | `false` |
| left-width  | 指定左侧滑动区域宽度                | `number`  | `auto`  |
| right-width | 指定右侧滑动区域宽度                | `number`  | `auto`  |

##### Slots

| 名称    | 说明           |
| ------- | -------------- |
| default | 自定义显示内容 |
| left    | 左侧滑动内容   |
| right   | 右侧滑动内容   |

##### Events

| 事件名 | 说明                        | 回调参数                                       |
| ------ | --------------------------- | ---------------------------------------------- |
| click  | 点击内容时触发              | -                                              |
| close  | 设置异步条件 `async` 时触发 | 异步关闭 (`clickPosition`, `instance`, `name`) |

##### close Params

| 参数名        | 说明                                               | 类型     |
| ------------- | -------------------------------------------------- | -------- |
| clickPosition | 关闭时的点击位置 (`left` `right` `cell` `outside`) | `string` |
| instance      | SwipeCell 实例，用于调用实例方法                   | `object` |
| detail        | 额外信息，包含 name 字段                           | `object` |

##### Methods

通过 ref 可以获取到 SwipeCell 实例并调用实例方法

| 方法名 | 参数            | 返回值 | 介绍             |
| ------ | --------------- | ------ | ---------------- | ---------------- |
| open   | position: `left` `right` | -                | 打开单元格侧边栏 |
| close  | -               | -      | 收起单元格侧边栏 |
