# Search 搜索框

### 安装

:::card

```javascript
import { Search } from '@okee-uikit/m-vue';

Vue.use(Search);
```

:::

### 代码演示

:::card

```topic
基础用法
```

```desc
通过 `v-model` 绑定搜索框的值
```

```html
<byted-search v-model="value" />
```

```javascript
data() {
  return {
    value: ''
  }
}
```

:::

### API

##### Props

| 参数         | 说明                                                           | 类型              | 默认值       |
| ------------ | -------------------------------------------------------------- | ----------------- | ------------ | --- |
| value        | 当前输入的值                                                   | `string` &#124; `number` | -   |
| clearable    | 是否启用清除控件                                               | `Boolean`         | `true`       |
| cancelable   | 是否显示取消按钮                                               | `Boolean`         | `false`      |
| borderRadius | 圆角值的大小，支持输入`px` `rem`，或可直接输入数字表示`px`大小 | `Number` `String` | `0`          |
| placeholder  | 填写输入字段的提示                                             | `String`          | `请输入内容` |
| autofocus    | 在页面加载时是否获得焦点                                       | `Boolean`         | `false`      |

##### Events

| 事件   | 说明                               | 回调参数            |
| ------ | ---------------------------------- | ------------------- |
| input  | 输入框内容变化时触发               | value: 输入框当前值 |
| search | 确定搜索, 当点击键盘确认按键时触发 | -                   |
| cancel | 取消搜索                           | -                   |
| focus  | 输入框获得焦点时触发               | event: Event        |
| blur   | 输入框失去焦点时触发               | event: Event        |
| clear  | 点击清除按钮时触发                 | event: Event        |
| click  | 点击时触发                         | event: Event        |

##### Slots

| 名称  | 说明                 |
| ----- | -------------------- |
| left  | 自定义搜索框左侧内容 |
| icon  | 自定义图标           |
| right | 自定义搜索框右侧内容 |
