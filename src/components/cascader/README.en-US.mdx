# cascader

### Install

:::card

```javascript
import { cascader } from '@okee-uikit/m-vue';

Vue.use(cascader);
```

:::

### Usage

:::card

```topic
Type
```

```html
<byted-cascader :options="options" v-model="selected" @change="handleChange" />
```

:::

:::card

```topic
Add collapse
```

```html
<byted-cascader :options="options" v-model="selected" collapse @change="handleChange" />
```

:::

:::card

```topic
Async loading children
```

```html
<byted-cascader
  :options="options2"
  v-model="selected"
  collapse
  @active-item-change="handleActiveItemChange"
  @change="handleChange"
/>
```

:::

### API

##### Props

| Attribute    | Description                          | Type      | Default |
| ------------ | ------------------------------------ | --------- | ------- |
| options      | cascader options array               | `array`   | -       |
| value        | selected object array（selectArray） | `array`   | []      |
| collapse     | Whether to collapse first column     | `boolean` | `false` |
| selectedItem | default selected value array         | `array`   | []      |

##### Events

| Event              | Description                     | Arguments                 |
| ------------------ | ------------------------------- | ------------------------- |
| change             | selected label array            | array（selectlabelArray） |
| input              | selected allProperty path array | array（selectArray）      |
| active-item-change | selected allProperty path array | array（selectValueArray） |
