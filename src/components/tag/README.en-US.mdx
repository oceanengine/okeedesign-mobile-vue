# Tag

### Install

:::card

```javascript
import Vue from 'vue';
import { Tag } from '@okee-uikit/m-vue';

Vue.use(Tag);
```

:::

## Usage

:::card

```topic
Type
```

```html
<byted-tag>Default</byted-tag>
<byted-tag text="primary" type="primary" />
<byted-tag text="success" type="success" />
<byted-tag text="warning" type="warning" />
<byted-tag text="danger" type="danger" />
```

:::

:::card

### API

##### Props

| Attribute  | Description                                                | Type      | Default   |
| ---------- | ---------------------------------------------------------- | --------- | --------- | --- |
| type       | Type, can be set to `primary` `success` `danger` `warning` | _string_  | `default` |
| plain      | Whether to be plain style                                  | _boolean_ | `false`   |
| round      | Whether to be round style                                  | _boolean_ | `false`   |
| size       | Size, can be set to `large` `medium`                       | _string_  | -         |
| color      | Custom color                                               | _string_  | -         |
| text-color | Text color                                                 | _string_  | `white`   | -   |

##### Slots

| Name    | Description  |
| ------- | ------------ |
| default | Default slot |

##### Events

| Event | Description                     | Arguments    |
| ----- | ------------------------------- | ------------ |
| click | Triggered when clicked          | event: Event |
| close | Triggered when click close icon | -            |
