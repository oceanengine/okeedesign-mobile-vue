# Picker 选择器

### 安装

:::card

```javascript
import { Picker } from '@okee-uikit/m-vue';

Vue.use(Picker);
```

:::

```topic
基础用法
```

```desc
单列`value`类型可为 `String` or `Number`
单列`options`数组中的每一项为`Object`, key值为`label`、`value`
```


```topic
多列选择
```

```desc
多列`value`类型可为数组, 数组每一项为 `String` or `Number`
多列`options`为二维数组, 二级数组中的每一项为`Object`, key值为`label`、`value`
```

```html
<byted-picker title="标题" v-model="value" :options="options" />
```

```javascript
export default {
  data() {
    return {
      value: [1, 1],
      options: [
        [
          { label: 0, value: 0 },
          { label: 1, value: 1 },

          { label: 3, value: 3 },
          { label: 4, value: 4 },
          { label: 5, value: 5 },
          { label: 6, value: 6 },

          { label: 0, value: 0 },
          { label: 0.5, value: 0.5 },

          { label: 2, value: 2 },
```

```topic
搭配弹层使用
```

```html
<byted-cell title="地域" value="设置" is-link :border="false" @click="show=!show" />
<byted-popup v-model="show" position="bottom">
  <byted-picker
    title="地域"
    v-model="value"
    :options="options"
    @cancel="show=!show"
    @confirm="confirm"
  />
</byted-popup>
```

```javascript
export default {
  methods: {
    confirm() {
      this.show = !this.show;
      this.$toast(`the value : ${JSON.stringify(this.value)}`);
    },
  },
};
```

### API

##### Props

| 参数                | 说明                                 | 类型                        | 默认值 |
| ------------------- | ------------------------------------ | --------------------------- | ------ |
| value               | 选项对象中的值                       | _string \| number \| array_ | `''`   |
| options             | 单列，级联时为数组, 多列时为二维数组 | _array_                     | `[]`   |
| title               | 顶部栏标题                           | _string_                    | -      |
| show-toolbar        | 是否显示顶部栏                       | _boolean_                   | `true` |
| confirm-button-text | 确认按钮文字                         | _string_                    | `确认` |
| cancel-button-text  | 取消按钮文字                         | _string_                    | `取消` |
| transition-duration  | 过度的时长                       | _number_                   | `600` |
| swipe-duration  | 惯性滑动动画的时长                    | _number_                   | `600` |

##### Picker Events

| 事件名  | 说明                 | 回调参数            |
| ------- | -------------------- | ------------------- |
| change  | 滚动结束时触发的事件 | 当前选中项的`value` |
| input   | 滚动结束时触发的事件 | 要选中项的`value`   |
| confirm | 点击完成按钮时触发   | Event               |
| cancel  | 点击取消按钮时触发   | Event               |

##### Slots

| Name  | Description               |
| ----- | ------------------------- |
| title | Custom title              |
| left  | Custom left side of header  |
| right | Custom right side of header |
