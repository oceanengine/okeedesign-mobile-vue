# Loading

### Install

:::card

```javascript
import { Loading } from '@okee-uikit/m-vue';

Vue.use(Loading);
```

:::

### Usage

:::card

```topic
Basic
```

```html
<byted-loading />
```

:::

:::card

```topic
Style
```

```html
<byted-loading type="circle" />
```

:::

:::card

```topic
Words
```

```html
<byted-loading text="loading..." />
```

:::

:::card

```topic
Vertical
```

```html
<byted-loading vertical>loading...</byted-loading>
```

:::

:::card

```topic
Icon Size
```

```html
<byted-loading size="1.2rem" />
```

:::

### API

##### Props

| Attribute | Description                                          | Type                 | Default   |
| --------- | ---------------------------------------------------- | -------------------- | --------- |
| type      | Can be set to `default` `circle`                     | `string`             | `default` |
| vertical  | Whether to arrange icons and text content vertically | `boolean`            | `false`   |
| size      | Icon size                                            | `string` or `number` | `''`      |

##### Events

| Event | Description | Arguments |
| ----- | ----------- | --------- |
| click | when click  | event     |
